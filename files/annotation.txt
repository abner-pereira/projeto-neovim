--------------------------------------------------
Movimentação no arquivo.
Comandos:
h	left
j	down
k	up
l	right

--------------------------------------------------
Salvar e sair do arquivo. 
Comando >> ZZ

--------------------------------------------------
Deletar um caracter.
Comando >> x

Deletar uma linha.
Comando >> dd

Deletar uma quebra de linha.
Comando >> J

--------------------------------------------------
Desfazer uma ação (Voltar).
Comando >> u

Refaz uma ação desfeita (Avançar).
Comando >> Ctrl + R

--------------------------------------------------
Inserir um novo texto ANTES a posição do CURSOR.
Comando >> i

Apendar um novo texto APÓS a posição do CURSOR.
Comando >> a

Inserir uma nova linha ABAIXO da posição do CURSOR e ativar o modo de INSERÇÃO.
Comando >> o

--------------------------------------------------
Repetição X vezes de uma comando.
Comando >> 3d (Exemplo: Deletar 3 linhas = dd + dd + dd)
